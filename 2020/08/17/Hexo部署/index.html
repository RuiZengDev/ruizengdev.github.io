Github部署

以下操作均在如下环境中进行：

系统环境：Oracle VM VirtualBox 新建的Windows 11虚拟机

  • 设备规格：

      产品 ID 00391-80000-00001-AA122
      系统类型 64 位操作系统, 基于 x64 的处理器

  • 系统参数：

    • 版本 Windows 11 专业工作站版
      版本 21H2
      安装日期 ‎2023/‎3/‎30
      操作系统版本 22000.194
      体验 Windows 功能体验包 1000.22000.194.0

网络环境：无代理环境（经过尝试，可以不需要代理但是需要耐心等待）

注册Github

  1. 打开Edge浏览器，地址栏输入网址：https://github.com ，点击Sign up进行注册。
  2. 输入你的对应信息，名称使用英文。

安装nodejs

  1. 打开Edge浏览器，地址栏输入网址：https://nodejs.org/en/download ，选择对应版本进行下载，我这里直接选择的是Windows Installer 中 64bit 安装包进行下载，小伙伴们可以根据自己操作系统选择安装不同版本的安装包。
  2. 双击你下载的安装包进行安装，安装比较简单，一路next即可，不懂的可以看视频。

安装Git

  1. 打开Edge浏览器，地址栏输入网址：https://git-scm.com/downloads ，选择对应版本进行下载，如果觉得下载速度太慢，可以用镜像链接：https://registry.npmmirror.com/binary.html?path=git-for-windows/，同样选择对应版本即可，我直接选择的是`64-bit Git for Windows Setup`，小伙伴们可以根据自己操作系统选择安装不同版本的安装包。

  2. 双击你下载的安装包进行安装，安装比较简单，不懂的可以看视频。

安装Visual Studio Code

为什么要安装一个Vscode呢，主要原因是后续我们需要编辑代码文件，使用Vscode操作简单，安装方便，使用传统文本编辑器效率过低。

  1. 打开Edge浏览器，地址栏输入网址：https://code.visualstudio.com/ ，选择对应版本进行下载。
  2. 以管理员身份进行安装即可。

检查安装是否完成

  1. win+r 输入cmd，输入node -v，npm -v，git -v返回对应版本号即为安装完成。
