连接使用

  1. 打开git bash，使用命令生成ssh公钥和私钥

    ssh-keygen -t rsa -C 'xxx@xx.com'，一路回车

  2. git bash使用命令查看生成的密钥

    cat ./.ssh/id_rsa.pub ，复制输出的内容到相应Git系统

添加多个Git密钥

  1. 生成一对密钥ssh key，for GitLab

  2. 生成一对密钥ssh key，for Github

  3. 在./ssh文件夹下新建文件名为config，用于管理多个ssh key。

    特别强调：

    这些配置允许您同时管理多个GitHub用户，并且不需要在每次连接时手动更改常规SSH配置。当您需要使用不同的GitHub目录时提供合适的主机别名即可完成SSH连接。例如：git clone git@github-user1:user/repo.git。
